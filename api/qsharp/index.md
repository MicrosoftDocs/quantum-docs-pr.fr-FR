# <a name="q-libraries"></a>Bibliothèques Q#

La bibliothèque standard Q# comprend des opérations, des fonctions et des types définis par l’utilisateur servant à développer des programmes quantiques. Elle comprend également trois autres bibliothèques : les bibliothèques de chimie, de mathématique et de machine learning.

Pour plus d’informations sur les bibliothèques Q#, consultez <xref:microsoft.quantum.libraries>.
