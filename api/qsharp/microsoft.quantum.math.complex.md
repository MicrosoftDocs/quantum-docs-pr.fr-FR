# <a name="complex-user-defined-type"></a>Type complexe défini par l’utilisateur

Espace de noms : [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Package : [Microsoft. Quantum. QSharp. Core](https://nuget.org/packages/Microsoft.Quantum.QSharp.Core)


Représente un nombre complexe par ses composants réels et imaginaires.
Le premier élément du tuple est le composant réel, le deuxième composant imaginaire.

```qsharp

newtype Complex = (Real : Double, Imag : Double);
```
