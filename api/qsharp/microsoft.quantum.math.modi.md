# <a name="modi-function"></a>ModI, fonction

Espace de noms : [Microsoft. Quantum. Math](xref:Microsoft.Quantum.Math)

Package : [Microsoft. Quantum. standard](https://nuget.org/packages/Microsoft.Quantum.Standard)


Retourne le modulo d’un nombre par rapport à un autre nombre.

```qsharp
function ModI (a : Int, b : Int) : Int
```


## <a name="input"></a>Entrée

### <a name="a--int"></a>r : [entier](xref:microsoft.quantum.lang-ref.int)

Entrée $a $ dont le modulo doit être retourné.


### <a name="b--int"></a>b : [entier](xref:microsoft.quantum.lang-ref.int)

Nombre par rapport auquel le modulo de $a $ doit être retourné.



## <a name="output--int"></a>Sortie : [int](xref:microsoft.quantum.lang-ref.int)

Le modulo $a \bmod b $.
