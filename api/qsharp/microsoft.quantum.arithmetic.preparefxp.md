# <a name="preparefxp-operation"></a>Opération PrepareFxP

Espace de noms : [Microsoft. Quantum. Arithmetic](xref:Microsoft.Quantum.Arithmetic)

Package : [Microsoft. Quantum. Numerics](https://nuget.org/packages/Microsoft.Quantum.Numerics)


Initialise un nombre à virgule fixe Quantum sur une constante classique.

```qsharp
operation PrepareFxP (constant : Double, fp : Microsoft.Quantum.Arithmetic.FixedPoint) : Unit is Adj + Ctl
```


## <a name="input"></a>Entrée

### <a name="constant--double"></a>constante : [double](xref:microsoft.quantum.lang-ref.double)

Constante dans laquelle initialiser le nombre à virgule fixe Quantum.


### <a name="fp--fixedpoint"></a>FP : [FixedPoint](xref:Microsoft.Quantum.Arithmetic.FixedPoint)

Nombre à virgule fixe (de type FixedPoint) à initialiser.
