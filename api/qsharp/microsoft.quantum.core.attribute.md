# <a name="attribute-user-defined-type"></a>Type défini par l’utilisateur de l’attribut

Espace de noms : [Microsoft. Quantum. Core](xref:Microsoft.Quantum.Core)

Package : [Microsoft. Quantum. QSharp. Core](https://nuget.org/packages/Microsoft.Quantum.QSharp.Core)


Attribut reconnu par le compilateur utilisé pour marquer les types définis par l’utilisateur en tant qu’attributs.
