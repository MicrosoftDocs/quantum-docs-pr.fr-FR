- <span data-ttu-id="38455-111">Fermionic Hamiltonians peut être représenté à l’aide d’intégraux à un et deux électrons.</span><span class="sxs-lookup"><span data-stu-id="38455-111">Fermionic Hamiltonians can be represented using one- and two-electron integrals.</span></span>
