<span data-ttu-id="47807-120">Plus important encore, la définition d’une Oracle de cette façon pour chaque État de base de calcul $ \ket{x}\ket{y} $ définit également la manière dont $O $ agit pour tout autre État.</span><span class="sxs-lookup"><span data-stu-id="47807-120">Importantly, defining an oracle this way for each computational basis state $\ket{x}\ket{y}$ also defines how $O$ acts for any other state.</span></span>
